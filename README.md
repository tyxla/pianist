Pianist
=======

## About

A great web app that allows you to play a warm, bright grand piano. Written (on JavaScript and pure CSS - no images) with love.

## How to play

You can play either by using the mouse, or by using the keyboard. Below you will find the default key mappings for the tones.

### Line 0 (ZXC...)
* C4 - Z
* D4 - X
* E4 - C
* F4 - V
* G4 - B
* A4 - N
* B4 - M
* C5 - ,
* D5 - .
* E5 - /

### Line 1 (ASD...)
* C#4 - S
* D#4 - D
* F#4 - G
* G#4 - H
* A#4 - J
* C#5 - L
* D#5 - ;

### Line 2 (QWE...)
* C5 - Q
* D5 - W
* E5 - E
* F5 - R
* G5 - T
* A5 - Y
* B5 - U
* C6 - I
* D6 - O
* E6 - P
* F6 - [
* G6 - ]

### Line 3 (123...)
* C#5 - 2
* D#5 - 3
* F#5 - 5
* G#5 - 6
* A#5 - 7
* C#6 - 9
* D#6 - 0
* F#6 - =
